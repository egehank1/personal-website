# egehankilic.com

Personal portfolio — built with Next.js 15, Tailwind CSS v4, and Framer Motion.

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Copy `.env.example` to `.env.local` and set `NEXT_PUBLIC_SITE_URL` to your domain.

## Stack

- Next.js 15 (App Router, Turbopack)
- Tailwind CSS v4
- Framer Motion
- TypeScript
